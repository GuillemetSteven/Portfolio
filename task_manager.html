Task Manager

Deferred Task Manager: program for scheduled execution of commands in UNIX/Linux

  • Dynamically provide command, delay between executions, and number of iterations
  • Avoid "time drift" by considering execution time and task management time

Expected work:

  • Zip archive with project directory, sources (.c and .h), makefile, README, and 1-2 page document
  • Names of team members, mode of work, task distribution, time spent, solution explanation, encountered difficulty and solution, project limits
  • Compilable, clean, and functional sources using only specified libraries
  • Delivery within corrector's deadlines

Improvements for final version:

  • Command-line mode ✅
  • Configuration file mode ✅
  • Single or multiple executions ✅
  • Choice of executable or execution directory ✅
  • Infinite repetition with log file and ability to stop ✅
